§4  Naming Convention

Format: object_verbed — snake_case, past tense.

Examples: install_clicked, email_submitted, scan_completed

Every event carries a product property. No prefixes in event names — segmentation happens via this property.

Multi-Product Scaling

Adding DiffGuard (or any future product) uses the exact same events with a different product value:

EventCommand CentralDiffGuard
install_clickedproduct: "command-central"product: "diffguard"
page_viewedproduct: "command-central"product: "diffguard"

§7  Adding a New Product

1
Add events to your site/app with product: "new-product" — same event names, new value
2
Add the events to METRICS.md event catalog
3
Filter the dashboard by the product property. That's it.
