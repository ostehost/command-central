Launch Features

Discovery + Grouping

`ps` scanning and Claude session-file detection auto-register active agents.

  • External terminal discovery (not VS Code-only)
  • Auto-group by project and git worktree
  • Agent badges for Claude, Codex, Gemini

Unified Sidebar State

Operational visibility for every running agent from one place.

  • Status colors: running/completed/failed/stopped/stuck
  • Session grouping by project
  • Resume-click for completed sessions

Lifecycle Controls

Control long-running sessions from the tree view and context menu.

  • Inline kill + restart actions
  • Stuck-agent heuristic detection
  • Backend switcher (Codex/Gemini) in status bar

Diff Intelligence

See what each agent changed before you open a PR.

  • Per-file touched list with +/- counts
  • Inline diff summary: `N files · +X/-Y`
  • Open agent output or git diff in one click

Notifications + Dock Signals

Know when work finishes even when your focus is elsewhere.

  • Rich completion/failure notifications with actions
  • Dock badge with running count
  • Dock bounce on completion/failure/stuck (macOS)
