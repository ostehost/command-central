Launch Features

Per-Project Emoji Dock Icons

Each project gets a distinct Dock identity that stays stable over time.

  • Auto-assign icon on first project launch
  • User override when you want a custom icon
  • Persistent identity across relaunches

Auto-Discovery in Any Terminal

The wedge: external-terminal agents appear in VS Code automatically.

  • Discovers Claude Code, Codex, and Gemini sessions
  • Supports Ghostty, iTerm2, Terminal.app, and tmux
  • No manual registration or naming rules

Agent Lifecycle Controls

Operate active agents from the VS Code sidebar without tab-hunting.

  • Kill and restart actions on each agent card
  • Open output and diff directly from sidebar actions
  • Works for auto-discovered and launcher-managed agents

Diff Summary Per Agent

Review change scope quickly before diving into full diffs.

  • Inline summary on each card: `N files · +X/-Y`
  • Per-file change list with additions/deletions per agent
  • Open Git diff for details with one click

Works With Your Terminal Stack

Use Ghostty Launcher when available, keep moving when it is not.

  • Ghostty, iTerm2, Terminal.app, and tmux compatibility
  • Graceful fallback to VS Code integrated terminal
  • Cross-repo prerelease gate with provenance
