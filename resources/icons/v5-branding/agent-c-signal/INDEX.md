## The Signal Concept at a Glance

### Working State (Searching)
- **Visual**: Amber waves radiating outward
- **Meaning**: Signal searching for changes in your repository
- **Color**: #f59e0b (light), #d97706 (dark)
- **Icon**: Expanding concentric arcs + center point

### Staged State (Acquired)
- **Visual**: Green lock with confirmation
- **Meaning**: Signal locked and transmission ready
- **Color**: #10b981 (light), #059669 (dark)
- **Icon**: Center point + lock brackets + verification ring
