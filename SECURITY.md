# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Command Central, please report it responsibly:

- **GitHub Issues**: [Open an issue](https://github.com/ostehost/command-central/issues) with the label `security`
- **Email**: security@oste.sh

Please include steps to reproduce the issue and any relevant details. We will acknowledge receipt within 48 hours and aim to provide a fix as quickly as possible.

## Supported Versions

Only the latest published version receives security updates.
